## Description
A smart street lighting system that uses a PIR motion sensor to detect movement and automatically turn street lights ON and OFF, reducing energy consumption.

## Project Image

![Smart Street Light](![Uploading implementation.png…])

## Components
- Arduino Uno
- PIR Motion Sensor
- LEDs
- Breadboard
- Jumper Wires

## Working
- Motion detected → Lights ON
- No motion detected → Lights OFF
